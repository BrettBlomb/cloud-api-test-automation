# Cloud API Test Automation Framework

This project demonstrates a Python-based API test automation framework using Pytest and AWS services.

## Features
- REST API test automation with Pytest
- Reusable API client
- AWS S3 integration for test artifacts
- CI-ready structure

## Tech Stack
- Python, Pytest
- REST APIs
- AWS (S3, EC2, Lambda)
- GitHub Actions (planned)

## Why this project
Built to demonstrate how QA automation can be designed to run reliably in cloud environments and integrated into CI/CD pipelines.
